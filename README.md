
# Roger - AI Discipline Coach

A chatbot AI assistant designed to help you build and maintain positive habits through personalized reminders, tracking, and motivational messaging.

## Features

- **Habit Tracking**: Monitor your daily habits and progress
- **Smart Reminders**: Receive timely notifications to stay on track
- **Personalized Coaching**: Get motivational messages tailored to your goals
- **Progress Analytics**: View your consistency and improvement over time
- **Flexible Goals**: Set and adjust habits based on your needs

## Getting Started

1. Clone the repository
2. Copy the /Server folder inside of a home-server or similar and add a .env file with your GEMINI API KEY
3. Install dependencies: `npm install`
3. Run the bot: `npm start`
4. Set your first habit and begin your discipline journey

## Usage

Chat with Roger to:
- Create new habits
- Log completions
- Request reminders
- Check your stats
- Get motivation

## Requirements

- Node.js 14+
- Internet connection
